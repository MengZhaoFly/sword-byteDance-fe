## sword-byteDance-fe

剑指字节跳动前端offer

## why byteDance

前端面试题仓库多如牛毛，只是因为比较喜欢字节的题目。
字节的题目，含蓄，优雅。
- 考基础：绝不浮光掠影，需要你最扎实的基础
- 考编码能力：绝不假大空，都是真实业务里最合理的需求
- 考算法：这个不多说，必备

## 最后

通过刷题拿到工作机会，往往是下下之策，一个合格的工程师应该注意平时工作的总结，反思，和长久以来的沉淀。共勉
